### About me

12+ years across **Payments, FinTech, Hospitality and Capital Markets** — combining hands-on **Quality Engineering** with **Product Ownership** and **Technical Project Management** in regulated, mission-critical environments.

Currently owning both QA and PO for **Planet's Digital Onboarding suite** — a regulated AML/UBO workflow. Built its API automation framework from scratch in Postman, and pioneered **AI-augmented QA** workflows using **MCP (Model Context Protocol)** connectors with Claude, ChatGPT, GitHub Copilot and Microsoft Copilot — letting LLMs drive Postman and Selenium directly, while keeping human judgement as the final gate on compliance-sensitive outputs.
